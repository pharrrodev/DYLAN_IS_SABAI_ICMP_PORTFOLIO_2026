# DYLAN_IS_SABAI_ICMP_PORTFOLIO_2026

Static audition / link-in-bio page: open `Start_here.html` locally, or deploy to Netlify.

## Deploy to Netlify from GitHub

1. **Push this repo** (any time you edit the site):

   ```bash
   cd DYLAN_IS_SABAI_ICMP_PORTFOLIO_2026
   git add -A
   git commit -m "Describe your change"
   git push
   ```

2. In **[Netlify](https://app.netlify.com/)**: **Add new site** → **Import an existing project** → **GitHub** → pick `DYLAN_IS_SABAI_ICMP_PORTFOLIO_2026` (if you have not already).

3. Build settings: **leave blank** — `netlify.toml` sets `publish = "."` (no build command).

4. **Updates:** With the site **linked to GitHub**, each `git push` to the default branch triggers a **new Netlify deploy** automatically. You do not edit anything separately on Netlify for HTML changes.

## Local preview

Double-click `Start_here.html` or `index.html`.
